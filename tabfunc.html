Funções das tags

table Representa uma tabela em HTML. Dentro do elemento
Th É o cabeçalho da tabela
Tr Cada linha de uma tabela é representada pela tag
Td Cada célula da tabela é representada pela tag
Thead Representar o cabeçalho da tabela
Tbody Representar o corpo da tabela
Tfood Representar o rodapé da tabela
Align Elemento de texto cetralizado
Border Adiciona uma borda na tabela como um todo.
Caption É a legenda da tabela.
Rowspan É o número de linhas que você quer que sejam ocupadas
Colspan Permite que uma célula ocupe várias colunas
Width Determina a largura da área de conteúdo de um elemento
